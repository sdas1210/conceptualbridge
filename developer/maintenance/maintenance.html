Maintenance Tools

Select a maintenance utility below. These tools are designed to inspect and process question TXT assets before they are used by the quiz system.

Citation Removal Engine

ACTIVE

Scans uploaded TXT question files and detects unwanted trailing citation blocks. Confirmed citations are removed while preserving the question structure and original content.

Main Functions
  • Upload and scan TXT question assets
  • Detect trailing citation blocks
  • Remove confirmed citations
  • Validate remaining citation count
  • Check question structure before output
  • Download cleaned TXT output

🔍

Proof Reader

ACTIVE

Character-validation and proofreading utility for identifying unexpected foreign-script characters inside English and Bengali question files.

Main Functions
  • Validate English and Bengali text
  • Detect unexpected Unicode characters
  • Show affected line and character
  • Navigate TXT files line by line
  • Edit and save corrected lines
  • Pass selected errors for the current session
  • Re-validate after corrections
  • Download the corrected TXT file
